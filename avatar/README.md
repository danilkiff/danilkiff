# Аватар GitHub

Семейство палитр одной композиции: светлое кольцо с ядром, цветные каналы
кромками, глитч-нарезка. Финал - ember: в профиль вручную грузится ember.png
(1024x1024), мастер - ember.svg.

## Быстрый старт

- набор палитр - словарь PALETTES в generate.py, файлы именуются <палитра>.svg/png;
- make svg - генерация SVG, make png - конвертация в PNG (инкрементально);
- make clean удаляет все svg и png в папке - посторонние файлы здесь не хранить;
- пересборка после правки палитр: make clean && make svg png.

## Ограничения композиции

- без SVG-фильтров и blend-режимов - переносимость рендереров;
- обязана выживать в круглом кропе и оставаться различимой в 20 px.
